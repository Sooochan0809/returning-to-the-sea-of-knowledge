# Returning to the Sea of Knowledge

## Overview

This repository contains code used in the participatory installation "Garden of Becoming" (2025, Osaka). It is returned here as part of an ongoing process of circulation.

---

## Concept

Open source knowledge is formed through countless contributions.
Fragments are received, transformed through context, and returned again.
This process forms a cycle:
receive
transform
return
—

## On Returning

This repository is not simply published. It is returned.
The code does not belong to a fixed point, but is placed within a flow, and changes through those who engage with it.

---

## Re-enactment (Score)

This act can be performed again.
For example:
take a fragment of code
transform it through your own context
publish (return) it to the network
This act may be accompanied by a ritual, or not. It will never be identical.

---

## Fork

Forking this repository creates branches.
These branches represent:
reinterpretations
divergences
visible relationships
